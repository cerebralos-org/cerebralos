# CerebraLOS Press Kit

## About

**CerebraLOS** is an open-source Cognitive OS for AI agents. Instead of perfect memory accumulation, it teaches AI how to forget and dream — making agents more context-aware and human-like.

- **GitHub:** https://github.com/cerebralos-org/cerebralos
- **npm:** https://www.npmjs.com/package/cerebralos
- **Version:** 1.0.4
- **License:** MIT

## One-liner

> The Cognitive OS that helps AI forget, dream, and evolve.

## Short Description (100 words)

CerebraLOS is a Git-native Layer of Subconsciousness for AI agents. While the industry races toward perfect memory, CerebraLOS takes the opposite approach: entropy-based Active Forgetting lets unimportant details decay naturally, while a nightly Sleep Job allows the AI to "dream" and consolidate surviving memories into higher-level insights. It runs 100% locally, requires no vector database, and integrates with any AI agent (Claude, Cursor, ChatGPT) via a minimalist Micro-MCP server. Built to forget. Designed to dream.

## Key Features

| Feature | Description |
|---|---|
| Active Forgetting | Entropy-based memory decay. Only what matters survives. |
| Sleep Job | Nightly cron job where AI dreams and consolidates memories. |
| Zero UI | No dashboards. Runs quietly in your terminal. |
| Micro-MCP | <500 token server for seamless agent integration. |
| Git-native | All memories stored as Markdown in a local Git repo. |
| LLM-agnostic | Works with any AI. No vendor lock-in. |

## Quick Start

```bash
npx cerebralos init
cerebralos sleep
cerebralos recall "what do you know about me?"
```
