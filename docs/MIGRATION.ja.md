# 既存のAIからCerebraLOSへの記憶移行ガイド

これまでChatGPTやClaudeに蓄積してきた「あなたに関する記憶」を、CerebraLOSに移行する方法を説明します。

## Step 1: 既存のAIから記憶を抽出する

以下のプロンプトをコピーして、現在お使いのAI（ChatGPT, Claude, Geminiなど）に送信してください。

```text
これまでの会話から学習した「私に関するすべての記憶とコンテキスト」をエクスポートしてください。特に指示や好みについては、可能な限り私の言葉をそのまま保持してください。

## カテゴリ（以下の順序で出力）:
1. **指示**: 今後従うよう明示的に求めたルール（トーン、フォーマット、スタイル、「常にXをする」「絶対にYをしない」、行動の修正など）。
2. **アイデンティティ**: 名前、年齢、場所、教育、家族、人間関係、言語、個人的な興味。
3. **キャリア**: 現在および過去の役割、会社、一般的なスキル領域。
4. **プロジェクト**: 私が意味を持って構築またはコミットしたプロジェクト。各プロジェクトにつき1つのエントリが理想です。何をするものか、現在のステータス、重要な決定事項を含めてください。
5. **好み**: 広く適用される意見、好み、働き方のスタイル。

## フォーマット:
各カテゴリに見出しを使用してください。各カテゴリ内で、1行に1つのエントリをリストし、古い日付順に並べ替えてください。各行は次のようにフォーマットします：
[YYYY-MM-DD] - エントリの内容

## 出力:
- コピーしやすいように、エクスポート全体を1つのMarkdownコードブロックで囲んでください。
```

## Step 2: CerebraLOSにインポートする

AIが出力したMarkdownコードブロックの中身をコピーし、ターミナルで以下のコマンドを実行します。

```bash
# クリップボードの内容をファイルに保存
pbpaste > ~/my_memory.md

# CerebraLOSにインポート
cerebralos import --from ~/my_memory.md --type ai_export
```

## Step 3: 夢を見る（記憶の統合）

インポートされた記憶は「周辺記憶（Peripheral Memory）」として保存されます。これをCerebraLOSのコアに統合するために、一度スリープジョブを実行します。

```bash
cerebralos sleep
```

これで完了です！あなたの過去の記憶はCerebraLOSの潜在意識に統合され、今後どのAIエージェントを使っても、この記憶が共有されるようになります。
